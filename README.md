# FallenSnipe
FallenSnipe | Merchant Detector / Sniper for Sols Rng

FallenSnipe is a macro created in Eon1-10 for the roblox game Sols Rng and is used to autobuy, detect and (eventually) snipe merchants.

Requirements:
1920x1080 100% Scale
Merchant Teleporter
Autohotkey V1.1 (On your system)
Tesseract (On your system)
